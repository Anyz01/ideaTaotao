## IDEA学习笔记

```
  ctrl + alt + B            快速进入实现类
  F2                        跳转到下一个高亮错误 或 警告位置
  ctrl + alt + v            自动填充变量
  ctrl + /                  这个是多行代码分行注释，每行一个注释符号
  ctrl + shift + /          这个是多行代码注释在一个块里，只在开头和结尾有注释符号
  Ctrl + Shift + Z          取消撤销 （必备）
  Ctrl + Shift + N          查找类 
  Ctrl + Alt + L            查找文件
```
